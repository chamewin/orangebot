# orangebot
Twitter Bot responds to positive and negative feedback on Twitter relating to keyword "homedepot". 
OrangeBot is made using a twitter API, IBM Watson Tone Analyzer, and nodemailer API. 
Twitter API will scan twitter for tweets containing keyword "homedepot" and feed the tweet into IBM's Tone Analyzer which 
will then determine if the tweet was positive or negative. In this case positive refers to level of joy detected while 
negative refers to the level of anger and disgust detected.
